UPUTSTVO ZA NOVE BOJE (tamno mornarsko plava + zlatna)
=========================================================

Paleta koja je primenjena:
- Pozadina (glavna):    #0A192F
- Pozadina (sekcije):   #112240  (svetlija nijansa, za naizmenicne sekcije)
- Tekst (glavni):       #CCD6F6  (bela sa plavicastim tonom)
- Tekst (opisni):       #8892B0  (priguseniji, za podnaslove/opise)
- Akcenat:              #F5A623  (zlatno-bakarna, dugmici/ikonice/cene)

Ove boje su definisane jednom u tailwind.config.ts kao:
accent, navy, navy-light, lightest, slate
- pa ako ikad zelis da promenis ton, menjas SAMO na tom jednom mestu.

OVAJ ZIP MENJA SLEDECE (sve sto je potrebno za boje):
- tailwind.config.ts   <- u ROOT folderu projekta (translator-website/)
- app/globals.css      <- zamenjuje postojeci
- app/layout.tsx       <- zamenjuje postojeci
- app/page.tsx         <- zamenjuje postojeci
- app/translations.ts  <- zamenjuje postojeci (tekst isti, samo radi sa novim kodom)
- app/components/*.tsx (svih 7 fajlova) <- zamenjuju postojece

TACNO GDE STA IDE (translator-website je tvoj postojeci folder na Desktopu):

  translator-website/
  ├── tailwind.config.ts      <-- zameni (u ROOT folderu, NIJE u app/)
  └── app/

          └── Footer.tsx       <-- zameni

NAJLAKSI NACIN (Windows):

1. Otvori ovaj ZIP (ne mora ekstrakcija, moze direktno iz ZIP-a)
2. Selektuj SVE iz ZIP-a (i "app" folder i "tailwind.config.ts")
3. Ctrl+C
4. Idi u Desktop\translator-website
5. Ctrl+V
6. Windows pita "Replace these files?" -> klikni
   "Replace the files in the destination"
   (translations.ts, layout.tsx, page.tsx, globals.css, tailwind.config.ts
    i sve iz components/ ce se zameniti - to je u redu, to je cilj)

7. U Command Prompt-u (u translator-website folderu):

   git add .
   git commit -m "Nova paleta boja - tamno plava i zlatna"
   git push

8. Sacekaj da Vercel zavrsi build (provericemo Build Logs ako ima greske),
   pa osvezi sajt (Ctrl+F5).

NAPOMENA: Ako i dalje imas STARI Pricing.tsx (sa cenama u dinarima vec
ubaceno) - ne brini, cene/tekst SU ostali isti, menjale su se SAMO boje
(className delovi), tekst i cene su identicni kao pre.
